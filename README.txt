Программа для симуляции маятника. Содержит в себе некоторые инструменты, 2 таблицы и сам маятник.

Использованные библиотеки/ресурсы:
* SFML 2.6.1 (https://www.sfml-dev.org/) - используется под лицензией zlib/png. Лицензия рилагается в файле LICENSE-SFML.txt
* Шрифт NotoSans - используется под лицензией SIL Open Font License (OFL). Лицензия прилагается в папке с шрифтом в файле OFL.txt

Для успешной компиляции необходим sfml 2.6. Подробнее про компиляцию программ с sfml см. в официальной документации библиотеки

Лицензия самого проекта - MIT, прилагается в файле LICENSE.txt

Приятного использования!
