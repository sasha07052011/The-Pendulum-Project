Программа для симуляции маятника. Содержит в себе некоторые инструменты, 2 таблицы и сам маятник.

Использованные библиотеки/ресурсы:
* SFML 2.6.1 (https://www.sfml-dev.org/) - используется под лицензией zlib/png. Прилагается в файле LICENSE-SFML.txt
* Шрифт NotoSans - используется под лицензией SIL Open Font License (OFL). Прилагается в папке с шрифтом в файле OFL.txt

Для запуска программы установите пакет libsfml-dev

Лицензия самого проекта - MIT, прилагается в файле LICENSE.txt

Приятного использования!
